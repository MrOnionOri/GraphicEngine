# Included voxel atlas

`Blocks.ppm` is the runtime 96x16 atlas. It contains six original 16x16 tiles: grass top, dirt,
stone, grass side, tree bark, and leaves. `BlocksOriginalV2.png` is the current compact preview.
The other PNG files preserve the generated source material.

The atlas was generated specifically for GraphicEngine and then downscaled with nearest-neighbor
sampling. It is not derived from Minecraft assets.
